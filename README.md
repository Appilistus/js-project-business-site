# AI Compass | AI Education for Children

Welcome to the **AI Compass** website project! This is a responsive landing page for an AI education platform designed for children, built to showcase a modern, engaging business site.

**AI Compass** is a fictional AI education platform for children, created as a concept for this project. The website is a landing page design where potential customers (parents and teachers) would ideally learn about the curriculum, understand the value of AI education, and sign up for trial classes.

This project was built as part of a coding bootcamp assignment to demonstrate skills in building responsive business websites.

The site is designed to be friendly, safe, and trustworthy, with a focus on:
- **Education Programs**: Tailored for different age groups (Elementary to High School).
- **Why AI**: Explaining the importance of AI literacy.
- **Safety**: Emphasizing a secure learning environment.
